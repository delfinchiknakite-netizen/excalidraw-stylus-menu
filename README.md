# Excalidraw Stylus Menu (S Pen)

Плагин-компаньон для [Obsidian Excalidraw](https://github.com/zsviczian/obsidian-excalidraw-plugin).
Делает рисование пером (Samsung S Pen) удобнее:

- **Жест-триггер пером → меню вставки блока** в точке касания: Текст, Стикер (текст в рамке),
  Фигуры (прямоугольник / эллипс / стрелка / линия), Встроенная заметка или изображение.
- **Касание у края блока → режим коннектора:** следующий тап по другому блоку рисует стрелку
  между ними.

Это отдельный плагин — он НЕ форкает Excalidraw, а вызывает его публичный `window.ExcalidrawAutomate`.

## Жест-триггер

По умолчанию — **касание пером по пустому месту** холста: коротко коснулись пером там, где нет
объектов, и подняли без движения → открывается меню. Если перо двигается — это обычное рисование;
касание по объекту — обычное поведение Excalidraw; касание у края блока — режим стрелки-коннектора.
Палец не перехватывается совсем.

Артефактная точка, которую карандаш может оставить от тапа, удаляется автоматически
(отключается в настройках).

Доступны и другие жесты (в настройках): **долгое нажатие**, **двойное касание пером**,
а также **боковая кнопка S Pen + касание** — но кнопка работает только если Android-WebView
устройства её отдаёт (проверяется через **Debug-оверлей**: ожидается `buttons=3`). На многих
Samsung веб видит только `buttons=1` — тогда боковая кнопка недоступна, используйте касание/нажатие.

## Кнопка на S Pen

Боковая кнопка пера часто **не доходит до веб-слоя** Obsidian (WebView отдаёт только `buttons=1`),
а BLE-кнопка «Air actions» вебу недоступна в принципе. Что делать:

1. **Проверьте через Debug-оверлей** (настройки плагина): жмите кнопку при наведении пера, при касании,
   делайте двойной клик. Лог покажет `hover/up/contextmenu/auxclick/keydown`. Если где-то всплывёт
   `b=2`/`b=3`, `button=5`, `contextmenu` или какая-то клавиша — кнопку можно привязать, напишите мне.
2. **Если не всплывает нигде** — используйте мост: в плагине есть команда **«Открыть меню вставки
   (стилус)»** (повесьте на неё горячую клавишу в *Настройки → Горячие клавиши*). Затем приложение-
   автоматизатор (**Tasker/MacroDroid**) ловит нажатие S Pen-кнопки и шлёт эту же клавишу — меню
   откроется в последней позиции пера.

## Сборка

```bash
cd excalidraw-stylus-menu
npm install
npm run build      # → main.js
```

## Установка

Скопируйте `main.js`, `manifest.json`, `styles.css` в папку вашего хранилища:

```
<vault>/.obsidian/plugins/excalidraw-stylus-menu/
```

Включите плагин в *Настройки → Сторонние плагины*. На телефон попадёт через Obsidian Sync /
git / облако (та же папка) или [BRAT](https://github.com/TfTHacker/obsidian42-brat).

## Известные ограничения (v0.1)

- Стрелка-коннектор рисуется между центрами блоков; точная привязка концов
  (`startObjectId`/`endObjectId`) зависит от версии Excalidraw и может не сработать —
  тогда стрелка просто рисуется по центрам.
- Жесты «долгое нажатие» / «двойное касание» могут оставить короткий штрих от пера,
  т.к. рисование начинается до распознавания жеста. Боковая кнопка от этого свободна.
